# SoluEscolIA

**HUB de soluções escolares com IA** — catálogo visual dos sistemas digitais desenvolvidos com o **Canva IA** pela turma do curso **IA na Educação** do **Senac Itabaiana**.

Reúne, em um só lugar, os sistemas criados pela turma, com responsáveis, áreas de uso e links de acesso.

## Tecnologia

Site estático em **HTML, CSS e JavaScript puro** — sem dependências e sem etapa de build.

- `index.html` — estrutura, cabeçalho e templates das telas
- `styles.css` — estilos, tema e responsividade
- `script.js` — dados dos sistemas, roteamento (hash) e interações

## Rodar localmente

Abra o arquivo `index.html` diretamente no navegador.
